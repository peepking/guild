# Guild Management MVP

ギルド運営シミュレーションゲームのMVP（Minimum Viable Product）です。
プレイヤーはギルドのマスターとなり、冒険者をスカウトし、クエストに派遣し、ギルドを大きくしていくことを目指します。

## 機能概要

*   **冒険者管理**: 酒場でのスカウト、ステータス確認、雇用。
*   **クエスト**: クエストの受注、冒険者パーティの編成と派遣、結果報告。
*   **施設運営**: ギルド内の施設管理（今後の拡張予定）。
*   **時間経過**: 1日ごとのサイクルでゲームが進行します。

## インストールと実行方法

開発環境で実行するには `Node.js` が必要です。

1.  **インストール**
    ```bash
    npm install
    ```

2.  **開発サーバーの起動**
    ```bash
    npm run dev
    ```
    ブラウザで表示されたローカルURL（通常は `http://localhost:5173`）にアクセスしてください。

3.  **テストの実行**
    ```bash
    npm run test
    ```

## 技術スタック

*   **言語**: JavaScript (Vanilla JS / ES Modules)
*   **ビルドツール**: Vite
*   **テストフレームワーク**: Vitest
*   **UI/スタイリング**: Vanilla CSS
